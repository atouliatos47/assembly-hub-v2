ASSEMBLY HUB - Quick Start
==========================

1. Double-click START.bat  (or run: python server.py)
2. Open browser and go to the URL shown in the console
3. Dashboard: http://YOUR-IP:3000/dashboard
4. Display:   http://YOUR-IP:3000/display/CENTER-ID

Requirements (already installed):
  pip install flask flask-sock openpyxl python-docx
